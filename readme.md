# Scrowl Editor

## Getting Started
### Requirements
- Node (Tested with v16.16.0)
- npm (Tested with v8.11.0)

#### Lerna

```
npx lerna

or

npm install --global lerna
```

## Init
### Bootstrap command runs npm init on all packages

```
cd ./
lerna bootstrap
```

## Building
### Builds
- Core templates
- Core themes
- Core blueprints
- Player
- Host
- Editor
- Electron platform builds

```
cd ./
./build.sh
```
