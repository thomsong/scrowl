# Scrowl Editor

## Getting Started
- Tested with Node v16.16.0
- Tested with npm v8.11.0

Requires Installing Lerna

```npm install --global lerna```


Bootstrap command runs npm init on all packages

```
cd ./
lerna bootstrap
```

## Building
### Builds
- Core templates
- Core themes
- Core blueprints
- Player
- Host
- Editor
- Electron platform builds

```./build.sh```
